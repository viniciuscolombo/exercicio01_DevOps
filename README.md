# exercicio01_DevOps

## DevOps
Conjunto de práticas que integra desenvolvimento e operações, com foco em automação, integração contínua e entrega contínua para acelerar e tornar mais confiável o processo de entrega de software.

## SRE
Abordagem que aplica princípios de engenharia de software à operação de sistemas, visando aumentar a confiabilidade, escalabilidade e disponibilidade por meio de automação, monitoramento e gestão de incidentes.

## Frontend
O frontend desta aplicação será desenvolvido utilizando a biblioteca React. O objetivo é construir uma interface de usuário interativa e responsiva para facilitar a interação com os serviços da aplicação.
